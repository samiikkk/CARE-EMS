# CARE-EMS: Towards Reliable Emergency Response Planning through Context-Aware Spatio-Temporal Graph Learning for EMS Demand Forecasting

We model Emergency Medical Service (EMS) demand across New York City (NYC) ZIP code zones as a spatiotemporal graph problem to capture inter-region spatial dependencies and complex temporal pattern, and introduce web based dashboard that shows CARE-EMS demand forecasts as ZIP code-level spatial demand views for EMS resouce planning.
